## Portfolio-V2
A web application from scratch to display skills and recent works

## Built with
* React
* React Router
* Styled-Components
* Deployed on Netlify

## Dev Setup
For local development

### Getting Started
Install and run
```
npm install
npm start
```
* Program will launch at: http://localhost:3000

Build the app for production 
```
npm run build
```
